Screen breakdown

Current Astra modules

Six primary modules form the current MVP and are already synced through shared data models.

01

Accounts

Add banks manually, set opening balances, and keep top-level assets trend responsive to transaction flow. Credit cards are currently fixed at zero utilization and ready for connector upgrade.

  • Bank picker with curated India bank list
  • Manual current balance and optional account mask input
  • Hold actions to manage or delete connected entries
02

Transactions

Transaction search and manual additions are integrated with category assignment. Android SMS mode can start from now to avoid legacy clutter during onboarding.

  • Search across merchant, category, and amount
  • Manual add form with category chooser
  • Fresh-start button for clean SMS-based tracking
03

Dashboard

A concise command center showing available budget, review queue, category utilization, and upcoming due items with consistent visual language.

  • Budget-left card with trend chart
  • Review list with category pills
  • Upcoming obligations and income surfaces
04

Categories

Category budgets support grouped hierarchies, live spend bars, and ring progress with subcategory expansion. Rebalance and add flows are built into the same surface.

  • Food and Shopping dropdown groups
  • Color-coded spend tone and progress states
  • Add category and rebalance budget forms
05

Recurring

Recurring cards can be marked paid and immediately reflected across categories and transactions. Add/edit flows support icon and category assignment.

  • Paid-state progress and totals
  • Long-press edit pattern for fast correction
  • Category intelligence with manual override support
06

Investments

Performance card, range toggles, top movers rail, and account return list give users a quick snapshot of momentum without overwhelming detail.

  • Range-based trend views
  • Top movers horizontal surface
  • Linked account return panel
