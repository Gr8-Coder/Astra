# Astra

Astra is a React Native financial planning app built with Expo.

## Current stack

- Expo SDK 54
- React Native
- Supabase
- TypeScript

## Features in this MVP

- Accounts
- Investments
- Transactions
- Dashboard
- Categories
- Recurring payments
- Email/password auth
- Google sign-in flow wired through Supabase

## Local development

```bash
npm install
npx expo start
```

## Backend

Supabase setup and migrations live in [`supabase/`](./supabase).
